The code is written in python3, it contains a main function, two functions to read from csv input with the data(movies.csv, rating.csv), and three distance calculation implementation, and three helper functions along the way.

To run, just type:
python3 recsystem.py

and the main function will interact with the user. If you are a first time user, please say yes to the returning user, if not please say no
